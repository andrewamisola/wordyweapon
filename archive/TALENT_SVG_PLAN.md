# Talent SVG Art Plan

## Art Style (Matching Weapons)
- **ViewBox**: 100×130 (same as weapons)
- **No bold outlines**: Shapes defined by fills, not strokes
- **Soft gradients**: linearGradient and radialGradient
- **Layered shapes**: Multiple overlapping elements for depth
- **Opacity variations**: 0.1-0.8 for soft, ethereal look
- **Highlight layers**: White shapes at 0.2-0.4 opacity
- **Shadow layers**: Dark shapes at 0.1-0.2 opacity

## Color Palette by Category
- **Generators (+W)**: Gold/Amber (#d4af37, #c9a227, #8b6914)
- **Multipliers (×M)**: Purple/Violet (#9333ea, #7c3aed, #581c87)
- **Retriggers (REREAD)**: Cyan/Teal (#06b6d4, #0891b2, #0e7490)
- **Converters**: Green (#22c55e, #16a34a, #15803d)
- **Thresholds**: Orange/Red (#f97316, #ea580c, #c2410c)
- **Effects**: Pink/Magenta (#ec4899, #db2777, #be185d)

## Icon Design Principles
1. **Clear silhouette**: Recognizable at small sizes
2. **Central focal point**: Main element in center
3. **Thematic connection**: Visual relates to talent name/effect
4. **Rarity indication**: T3 icons more elaborate, T1 simpler

## SVG Template Structure

```svg
<svg viewBox="0 0 100 130" xmlns="http://www.w3.org/2000/svg">
  <defs>
    <!-- Gradients defined here -->
    <linearGradient id="talent-main" x1="0%" y1="0%" x2="100%" y2="100%">
      <stop offset="0%" stop-color="MAIN_COLOR"/>
      <stop offset="100%" stop-color="SHADOW_COLOR"/>
    </linearGradient>
    <radialGradient id="talent-glow" cx="50%" cy="50%" r="50%">
      <stop offset="0%" stop-color="GLOW_COLOR" stop-opacity="0.8"/>
      <stop offset="100%" stop-color="GLOW_COLOR" stop-opacity="0"/>
    </radialGradient>
  </defs>

  <!-- Background glow (outer) -->
  <ellipse cx="50" cy="65" rx="45" ry="55" fill="url(#talent-glow)" opacity="0.3"/>

  <!-- Main icon shape -->
  <g id="talent-main-shape">
    <!-- Primary shape with gradient fill -->
    <!-- Highlight layer (white, low opacity) -->
    <!-- Shadow layer (dark, low opacity) -->
  </g>

  <!-- Detail elements -->
  <g id="talent-details">
    <!-- Secondary shapes, accents -->
  </g>

  <!-- Outer subtle ring (optional for T3) -->
  <circle cx="50" cy="65" r="40" fill="none" stroke="ACCENT_COLOR" stroke-width="1" opacity="0.2"/>
</svg>
```

## Code Structure (TALENT_SVG Object)

```javascript
const TALENT_SVG = {
  // T1 Common - Simple designs
  steady_hand: `<svg viewBox="0 0 100 130">...</svg>`,
  first_blood: `<svg viewBox="0 0 100 130">...</svg>`,
  // ... etc

  // T2 Uncommon - Medium complexity
  world_sky_focus: `<svg viewBox="0 0 100 130">...</svg>`,
  // ... etc

  // T3 Rare - Elaborate designs
  anaphora: `<svg viewBox="0 0 100 130">...</svg>`,
  // ... etc
};

// Usage: Similar to WEAPON_SVG, render in talent card
function renderTalentIcon(talentId) {
  return TALENT_SVG[talentId] || TALENT_SVG.default;
}
```
